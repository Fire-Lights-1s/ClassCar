# classcar

**요약**

- 고령층의 유휴차량을 대여 할 수 있는 앱을 만들기

**사용 언어 & 환경**

- Dart
- Flutter
- firebase
- kakao developers
- google cloud

**시기**

- 캡스톤 디자인 2023.3 ~ 2023.11

## 앱의 목적

- 고령층의 유휴 차량을 젊은 층이 대여해서 사용할 수 있는 서비스를 만드는 것
- 고령층의 유휴 차량을 대여해주면서 고령층의 수입이 될 수 있음
- 젊은층에게 이동수단의 선택폭을 넓혀줄 수 있음

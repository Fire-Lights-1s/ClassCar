# classcar

졸업과제로 flutter와 firebase로 차량 대여 서비스에서 차량 등록자의 앱을 만들었습니다.

## 프로젝트 소개
* 기간 : 2023.04 ~ 2023.09
* 환경 : flutter, Dart, firebase
* 사용 API : google map API, kakao 지도 API
  
 ### 팀원
 * 유진호 : 로그인, 회원가입, 마이페이지, 각 페이지 통합, PPT 및 영상 제작
 * 우병섭 : 로그인, 차량관리 페이지, DB 연결
 * 박규영 : 요청내역 페이지
### 개발 배경
* 현재 대한민국의 인구 구조를 50~60대의 비율이 높고 미래에는 노령층이 인구 비율의 대부분을 차지할 것입니다. 
* 대한민국 교통사고 사망자 비율에서 4명중 1명은 고령층 운전 사고로 밝혀졌습니다. 
* 정부와 자치단체 등이 다양한 제도를 실시하여 고령층이 면허를 자진 반납하도록 유도하고 있지만 운전면허를 반납하는 고령층 비율은 낮은 것이 현실입니다. 
* 지금 상황에서 고령층 유휴 차량을 이용하여 수익을 늘리는 차량 공유 앱이 있다면 면허의 반납 비율 증가에도 긍정적인 영향이 있을 것입니다.

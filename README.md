# classcar

졸업과제로 flutter와 firebase로 차량 대여 서비스에서 차량 등록자의 앱을 만들었습니다.

## 프로젝트 소개
* 기간 : 2023.04 ~ 2023.09
* 환경 : flutter, Dart, firebase
* 사용 API : google map API, kakao 지도 API
  
 ### 팀원
 * 유진호 : 로그인, 회원가입, 마이페이지, 각 페이지 통합, PPT 및 영상 제작
 * 우병섭 : 로그인, 차량관리 페이지, DB 연결
 * 박규영 : 요청내역 페이지
